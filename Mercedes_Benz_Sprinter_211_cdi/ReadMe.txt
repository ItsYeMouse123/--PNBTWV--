----INSTALLATION FOR REPLACE MODEL-----

1. Using OpenIV navigate to the following location:
		mods > update > x64 > dlcpacks > patchday5ng > dlc.rpf > x64 > levels > gta5 > vehicles.rpf
2. Replace all 4 files from provided 'Replace' folder - rumpo.yft, rumpo_hi.yft, rumpo.ytd and rumpo+hi.ytd

3. Save and navigate to the next location for working dials:
		mods > common.rpf > data > levels > gta5
4. Open 'vehicles.meta', search for '<modelName>rumpo' scroll down a bit until you see '<dashboardType>VDT_TRUCK</dashboardType>' and replace it with '<dashboardType>VDT_GENTAXI</dashboardType>'. Click Save.

5. Navigate to the last location:
		mods > update > update.rpf > common > data > levels > gta5
6. Repeat step 4
